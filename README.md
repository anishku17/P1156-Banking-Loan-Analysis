# 📊 Banking Loan Analysis – Data Analyst Project  

## 📌 Overview  
This project analyzes bank loan data to track key banking KPIs such as Loan Approval Rate, Total Funded Amount, Default Rate, and Customer Risk Segmentation using SQL and Power BI.

## 🛠 Tools Used  
- Excel – Data cleaning and preprocessing  
- SQL – Data querying and KPI calculation  
- Power BI – Interactive dashboard creation  
- Tableau – Additional visualization and reporting  

## 📊 Key Insights  
- Identified overall loan approval and rejection trends  
- Analyzed default rate based on income and employment type  
- State-wise loan distribution visualized  
- High-risk customer segments detected  
- Loan amount vs annual income relationship evaluated
